Countdown setup

To setup the countdown, open "main.js" file in the "js" folder and find the following line of code there (line 43):

$('#counter').countdown('2015/11/16 12:00:00').on('update.countdown', function(event) {

Now just change the date (2015/11/16) and time (12:00:00) to the date-time of you liking and that's it.

Gradient Background setup

There are 20 gradient background css classes in your "style.css" file in the "css" folder. They are named like- .gradient-1.... .gradient-20. For gradient layer opacity [transparency] there are built in classes like - .opacity-50, .opacity-90 etc. In"gradient.html" file find the following code and replace the relevant classes with other available classes in your "style.css" file:

class="gradient-overlay gradient-1 opacity-30"

Multiscroll background setup

To setup the split backgrounds, open "style.css" file in the "css" folder and find the following line of code there (line 167):

#left1 {
	background: url('http://placehold.it/960x1080')no-repeat top right;
	background-size: cover;
}

#right1 {
	background: url('http://placehold.it/960x1080')no-repeat top left;
	background-size: cover;
}

#left2 {
	background: url('http://placehold.it/960x1080')no-repeat top right;
	background-size: cover;
}

#right2 {
	background: url('http://placehold.it/960x1080')no-repeat top left;
	background-size: cover;
}

#left3 {
	background: url('http://placehold.it/960x1080')no-repeat top right;
	background-size: cover;
}

#right3 {
	background: url('http://placehold.it/960x1080')no-repeat top left;
	background-size: cover;
}

The system is based on your pictures splitted in two parts. For instance, take a picture of 1920x1080. Split it into 2 parts; each having 960x1080 dimensions. Now replace the image paths in the css (see above).

Google Maps setup

To setup the Google Maps, open "main.js" file in the "js" folder and find the following lines of code there (line 105 and 131):

// Map center
var latlng = new google.maps.LatLng(51.498609000000000000,-0.133906000000024500);
// Marker Position
var myLatlng = new google.maps.LatLng(51.498609000000000000,-0.133906000000024500);

Edit the decimal numbers separated by coma within the first bracket. the first number is Latitude and the secon one is Longitude of the location. You can place the marker on a different position (currently the center) on the map by choosing different Latitude & Longitude.

Newsletter Mailchimp / File / Email

With this Coming soon, you have the choice between 3 solutions to know the persons wanting to be alerted when your website will be online, the launching of your event etc...

1. Mailchimp - Newsletter service

  • More than 6 million people use MailChimp to create, send, and track email newsletters. Whether you're self-employed, you manage projects for clients, or you work for a Fortune 500 company, MailChimp has features and integrations that will suit your email marketing needs. More informations
  • For the configuration with Mailchimp, you must just to know your API Key and the ID of your list.
  • API Key
  • 1- Click your profile name to open the Account Panel and choose Account Settings.

  • 2- Click the Extras drop-down menu and choose API keys.

  • 3- Copy an existing API key or click the Create A Key button.

  • 4- Label your key to keep your keys organized.



  • List ID
  • 1- Navigate to the Lists page on the left of your dashboard.
  • 2- Click the list's drop down menu and select Settings.

  • 3- Scroll down to the bottom of the Settings page to find the unique List ID for the list.

Twitter widget setup

To setup the Twitter widget, open "twitter.html" file and find the following lines of code at the bottom of the page :

$(document).ready(function () {

    twitterFetcher.fetch('521192861014835200', 'twitter', 1, false, false, true, 'default', false);

});
 
Replace the generic [521192861014835200]widget ID with your real widget ID. The number "1" is the maximum number of tweets you want to show. The first boolean string is to link or not hyperlinks on the tweet. The second boolean is to show photos. The third boolean is to show tweet date. And done!

Contact form setup

To setup the contact form, open "contact_me.php file in the "php" folder and find the following lines of code there:

$to = 'myemail@mail.com';
 
Just replace it with the email ID where you would like the contact form emails to be sent to.
